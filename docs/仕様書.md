# 概要
- ヒットアンドブロー型英単語当てゲーム
- CLIゲーム（Windows PowerShell）
- 4桁の実在する単語

# ゲームの流れ
## スクリプトを実行したら，スタート演出を表示。
- ゲームが始まったこと
- 文字数

## インゲーム
- 入力を促す
- ヒット，ブローの数表示
- 繰り返し

## 成功時
- 成功演出

# ヒット＆ブローの判定アルゴリズム
- ヒット：場所と文字一致
- ブロー：文字一致
- 文字の重複ありの場合
    - 解答：baby, 回答：bike, 1ヒット，0ブロー
    - 解答：baby, 回答：bomb, 1ヒット，1ブロー
    - 解答：baby, 回答：xbxx, 0ヒット，1ブロー
    - 解答：baby, 回答：xbxb, 0ヒット，2ブロー
